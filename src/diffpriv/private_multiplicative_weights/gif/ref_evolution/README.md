# Reference evolution

This evolution is the one on the database that is generated randomly.
It uses the List. implementation.
We see that it converges.

The parameters are the following:

|   Parameter   |    Value   |
|:--------------|-----------:|
| $\varepsilon$ |        $1$ |
| $\alpha$      |    $1/100$ |
| $\beta$       |    $1/100$ |
| $\eta$        | $\alpha/2$ |
